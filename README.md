# Программа "Построение «Цветка Жизни»"

Название программы: **Flower_of_Life, версия 1.2**

Автор: **Михаил Качаргин**

## Описание

Программа визуализирует геометрический узор «Цветок Жизни» — сакральный символ, состоящий из множества перекрывающихся 
окружностей и лепестков, образующих симметричную структуру. Реализована с использованием модуля turtle в Python.

Происхождение символа окутано тайной, но считается, что он был создан около 6000 лет назад в древней Месопотамии. 
`astromeridian.ru`

**Некоторые факты из истории символа:**
- Изображения «Цветка Жизни» найдены в Египте, в Храме Осириса (город Абидос). Там его высекли в граните, 
вероятнее всего, в качестве ока бога Ра — символа власти фараонов.
- Цветок Жизни встречали учёные, изучающие Финикию, Ассирию, Индию, Азию и Ближний Восток.
- Символ использовали представители средневекового искусства.
- Художники эпохи Возрождения активно применяли узор во время написания своих картин.
- Цветок Жизни обнаружили во многих местах, имеющих религиозное значение: гора Синай в Египте, места паломничества
в Тибете, индийские, испанские и греческие часовни, дворцы в Китае и Японии.
`author.today`, `wikihow.com`

## Функциональные возможности

- Построение отдельных лепестков (дуга + дуга).
- Формирование цветка из заданного числа лепестков.
- Рисование окружностей-границ.
- Создание симметричной композиции из нескольких цветков и четырёхлистников по кругу.
- Настройка цвета, радиуса и расположения элементов.

## Требования

- Python 3.x
- Стандартный модуль `turtle` (входит в поставку Python)

## Структура кода

### Основные функции

1. `petal(x_begin, y_begin, radius, corner, color="black")`

Рисует один лепесток из двух дуг.

**Параметры:**

- `x_begin`, `y_begin` — координаты начала лепестка.
- `radius` — радиус дуги.
- `corner` — угол наклона лепестка.
- `color` — цвет контура (по умолчанию чёрный).

**Возвращает:** координаты конца лепестка.

2. `flower(x, y, radius, total_places_petals, pattern_num=None, color="black")`

Строит цветок из лепестков (используя `petal`).

**Параметры:**

- `x`, `y` — координаты центра цветка.
- `radius` — радиус цветка.
- `total_places_petals` — общее число мест для лепестков.
- `pattern_num` — номер шаблона (0, 1, 2) для выбора 4 лепестков; если `None` — рисуются все.
- `color` — цвет контура.

**Возвращает:** кортеж координат концов лепестков.

3. `cir(x, y, radius, color="black")`

Рисует окружность с заданным центром и радиусом.

**Параметры:**

- `x`, `y` — координаты центра.
- `radius` — радиус окружности.
- `color` — цвет контура.

### Основная программа

- Настраивает окно черепашки `turtle`: размер, фон, скорость, толщину линии.
- Рисует внешние границы (две концентрические окружности).
- Создаёт лепестки‑ограничители по кругу.
- Строит центральный цветок и дополнительные лепестки вокруг него.
- Размещает 6 полных цветков и 6 четырёхлистников по окружности.

## Параметры по умолчанию

- Радиус лепестка/цветка: 80 пикселей.
- Число мест для лепестков в цветке: 6.
- Количество цветков по кругу: 6.
- Угол между элементами: 60 градусов (рассчитывается автоматически).

## Как запустить

1. Убедитесь, что установлен Python 3.x.
2. Сохраните код в файл `Flower_of_Life.py`.
3. Запустите скрипт: `python Flower_of_Life.py`.
4. В появившемся окне, черепашка нарисует фигуру «Цветок жизни».

## Настройка параметров

Чтобы изменить внешний вид фигуры, отредактируйте следующие переменные в секции **«Настройки построения цветка»**:

- `radius` — радиус лепестков и цветков.
- `total_places_petals` — число лепестков в цветке.
- `num_flowers` — количество цветков по кругу.
- `color` в вызовах функций — цвет элементов.
